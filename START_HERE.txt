🎯 WHAT THE AGENT DOES

For Each Grid Scenario:

1. ANALYZE
   Uses Groq LLM to assess grid stress
   Considers: frequency, demand surplus, renewable %

2. SELECT  
   Uses Groq LLM to identify best devices
   Considers: flexibility, capacity, temperature bounds

3. PLAN
   Uses Groq LLM to generate specific actions
   Creates: device ID, temperature target, expected MW reduction

4. VALIDATE
   Applies safety checks
   Ensures: temps 16-28°C, positive reduction, valid bounds

RESULT: List of DR actions for dashboard user to apply
